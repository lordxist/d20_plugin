planned_generation_is_valid?(assignment)

Checks whether the planned generation of ability scores is valid. Planned generation works as follows: Scores start at 8. The total amount of score points is reduced by a certain cost for each score increase. If the points fall below zero, the entire assignment is invalid.

This returns the remaining score points:

Ability.planned_generation_is_valid?(:str => 9, :dex => 8)

And this false:

Ability.planned_generation_is_valid?(:str => 10, :dex => 8)
